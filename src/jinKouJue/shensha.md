
### 常用吉神

| 神名 | 排法（口诀/规则） | 作用 |
| :--- | :--- | :--- |
| **天德** | 正丁二申庚，三壬四辛同，五癸亥六甲，七癸八甲寅，九丙十居乙，子巳丑庚中。 | 主化解百祸，忧危得安，逢凶化吉，尊长贵人吉庆。 |
| **天德合** | 课内见与天德相合之干（如天德为丁，课见壬）。 | 主解百祸，吉庆，化凶解忧。 |
| **月德** | 寅午戌月在丙，亥卯未月在甲，申子辰月在壬，巳酉丑月在庚。 | 主和睦，万事顺达，尊长吉庆，化凶为吉。 |
| **月德合** | 课内见与月德相合之干（如月德为丙，课见辛）。 | 作用同月德，吉庆程度稍次。 |
| **天赦** | 春戊寅，夏甲午，秋戊申，冬甲子。 | 主解刑禁、官司、危险之灾，利修造、婚姻。 |
| **天喜** | 春季见戌亥子，夏季见丑寅卯，秋季见辰巳午，冬季见未申酉。 | 主家中喜庆、婚姻、进财、宴会、升迁、添人口。 |
| **天马** | 正七午，二八申，三九戌，四十子，五十一寅，六腊辰。 | 主办事迅速，逃亡远去不归，失物难寻。 |
| **驿马** | 申子辰马在寅，亥卯未马在巳，巳酉丑马在亥，寅午戌马在申。 | 主求事迅速，升迁远行，喜庆欢乐；逢冲破则吉凶不定。 |
| **三奇** | 天三奇（甲戊庚），地三奇（乙丙丁），人三奇（壬癸辛）。 | 利见大人，百事吉昌，贵人相助，万事亨通。 |
| **天医** | 正月戌，二月亥，顺推至十二月酉。 | 问病忧中有乐，死而复生，得良医，危中有安。 |
| **生气** | 正月起子，二月起丑，顺推至十二月（对冲为死气）。 | 绝处逢生，开辟新事业，先难后易，万物萌发。 |
| **六甲** | 人元见甲。 | 主有不测之喜，婚姻财帛，官职文书之喜，为头目。 |


### 常用凶煞

| 神名 | 排法（口诀/规则） | 作用 |
| :--- | :--- | :--- |
| **劫煞** | 申子辰见巳，巳酉丑见寅，寅午戌见亥，亥卯未见申。 | 君子（官员）见之主权柄；常人见之主凶伤、官司、伤残。 |
| **截命灾煞** | 甲己见申酉，乙庚见午未，丙辛见辰巳，丁壬见寅卯，戊癸见子丑。 | 求谋不通，出行受阻，生产不顺，九死一生。 |
| **五鬼** | 甲己巳午癸未存，乙庚寅卯守黄昏，丙辛子丑来冲位，丁壬戌亥墓临门，戊癸忌占申酉位。 | 损财、官司、车祸、道路冤仇、出行大凶。 |
| **丧门** | 太岁（年支）前二辰（如戌年见子）。 | 主凶丧、孝服、哭泣、病灾、家有凶事。 |
| **吊客** | 太岁（年支）后二辰（如寅年见子）。 | 主阴私、凶伤、服毒自杀、亲朋凶丧。 |
| **丧车** | 春酉，夏子，秋卯，冬午。 | 主病灾、车祸、血光、产死、离别。 |
| **灭门** | 阴月前三位，阳月后三位（如未月为阴，前三戌为灭门）。 | 忌迁居、嫁娶、孕育，主病灾、官司、伤财。 |
| **天罗/地网** | 日支前一辰为天罗，对冲为地网。 | 主牢狱官司，失物难寻，阻隔不通。 |
| **关/隔/锁** | 酉上见木（寅）为关，卯上见土（戌）为隔，卯上见金（申）为锁。 | 关节不通，囚禁，远人不归，办事迟留。 |
| **往亡** | 立春后七日（如甲寅），之后依口诀类推。 | 忌拜官、上任、出行、出军、嫁娶，主路不通、全军灭。 |
| **四绝** | 寅酉（金绝）、卯申（木绝）、午亥（水绝）、子巳（火绝）。 | 主办事难成，劳而无功，口舌，伤损，车祸。 |
| **四败** | 水土遇酉，火遇卯，木遇子，金遇午。 | 主拘禁、口舌、斗讼、刑狱，不宜问病。 |
| **天盗** | 贵神或将神见“子”，或逢贼动。 | 主被盗、失财。 |
| **天鬼** | 春酉，夏子，秋卯，冬午。 | 主凶灾，鬼变，不测之灾。 |
| **飞廉** | 正月戌，二月巳，三月午，四月未，五月申，六月酉...顺推。 | 主迅速，非常惊骇，或喜或怒不测之事。 |
| **望门煞** | 劫煞的对冲之神（如劫煞在巳，亥为望门煞）。 | 主妄想、空想、奸淫，办事难成。 |
| **病符** | 太岁后一辰（如寅年见丑）。 | 主大病、灾祸。 |
| **官符** | 天乙贵神对冲之神。 | 主官司刑法，问病大凶。 |
| **六丁** | 人元见丁。 | 主家中不安宁，惊恐，火光，神经衰弱，伤六畜。 |
| **禄倒** | 甲年见卯，乙年见辰，丙戊年见午等（禄前一位）。 | 主百事不顺，官损事难成，病者大凶。 |
| **马倒** | 寅午戌年见酉，申子辰年见卯，巳酉丑年见子，亥卯未年见午。 | 主百事不顺，求事阻隔，不宜求官。 |
| **四丘** | 春丑，夏辰，秋未，冬戌。 | 主争论田土坟墓，争讼，病死。 |
| **四墓** | 春未，夏戌，秋丑，冬辰。 | 主争讼坟墓之事，问病凶，万物收藏。 |
